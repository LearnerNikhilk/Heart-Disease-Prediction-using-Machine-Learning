# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview
This project applies machine learning to predict the likelihood of heart disease based on clinical metrics (like age, cholesterol, and maximum heart rate). The goal is to build a reliable predictive model that can assist in early diagnosis.

## 🛠️ Tools & Technologies Used
* **Language:** Python
* **Data Manipulation & Analysis:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn (Logistic Regression)
* **Preprocessing:** StandardScaler

## 📊 Key Insights & Results
* Conducted Exploratory Data Analysis (EDA) to identify strong correlations between chest pain types, maximum heart rate, and heart disease presence.
* Built and trained a **Logistic Regression** model.
* Achieved an **Accuracy of 85.25%** and a **Precision of 87.10%**, demonstrating strong reliability in predicting positive cases.

## 📂 Repository Structure
* `Logistic_regression_3algo.ipynb`: The complete notebook containing EDA, data preprocessing, model training, and evaluation.
* `heart.csv`: The dataset used for training and testing.
